Project Overview

The Barangay Profiling System is a comprehensive web application designed to streamline the management of resident records for local government units. This system replaces manual record-keeping with a secure, efficient digital solution that includes features for resident registration, document requests, and reporting.

Key Features

  • Secure resident registration and profile management
  • Document request, approval, and tracking system
  • Automated report generation for barangay officials
  • Role-based access control for admins, staff, and officials
  • Data visualization dashboard for real-time barangay statistics
  • Integrated Twilio SMS notifications to inform residents of updates
  • Responsive design optimized for both desktop and mobile devices

Challenges & Solutions

One major challenge was ensuring data security while making the system accessible to barangay officials with varying technical skills. I implemented a robust authentication system with role-based permissions and created an intuitive interface with clear instructions for each process.

Project Overview

SansLechon is a full-stack e-commerce platform designed for selling traditional Filipino lechon and party food packages online. The platform allows users to browse offerings, place orders, and connect with partnered shops. Built with user convenience in mind, the system ensures a smooth online ordering experience for both customers and administrators.

Key Features

  • Dynamic product listings for lechon, bilao, and cakes
  • Shop-wise cart system with quantity controls and real-time updates
  • Email-based authentication with optional OTP login
  • Partner shop integration and individual shop management
  • Image storage via Laravel’s storage linking for product galleries
  • Responsive design tailored for mobile-first customers
  • Introductory freebies and promotions based on cart contents

Technical Highlights

The platform was built using Laravel and Livewire to deliver a highly interactive UI with minimal JavaScript. I implemented modular cart logic to handle multiple shops in a single checkout session, added a secure image upload system using Laravel’s file storage, and crafted dynamic sliders for showcasing shop-specific products. Performance optimization was also achieved through caching and eager loading techniques.

San's Lechon Affiliate System

HTML
CSS
Tailwind
JavaScript
QR Code Integration

Project Overview

The San's Lechon Affiliate System is a web-based platform designed to empower individuals to earn by promoting our lechon offerings and partner shop products online. Affiliates get access to their own dashboards, trackable referral links, QR codes for offline promotion, and commission summaries. Designed with a mobile-first approach, the system ensures seamless affiliate engagement and admin-level control over performance and payouts.

Key Features

  • Affiliate dashboard with real-time referral and commission stats
  • Custom referral link generator with auto-generated QR codes
  • One-click copy-to-clipboard functionality for easy sharing
  • Social media sharing integration (Facebook, Messenger, etc.)
  • Secure registration/login system with OTP email support
  • Mobile-responsive UI for on-the-go tracking and link sharing
  • Admin panel to manage affiliate users, commissions, and payouts

Technical Implementation

The affiliate system was built using Laravel and Livewire for real-time interactivity without page reloads. Clipboard API was used for seamless link copying, and QRCode.js was integrated to generate scannable codes dynamically. Chart.js powers the performance visualization in the affiliate dashboard. Referral tracking was implemented using unique user-based tokens, with secured route bindings and middleware to avoid abuse. The entire platform follows modern web accessibility standards (WCAG), and responsiveness was fine-tuned using Tailwind CSS.
